code[4,5,7,8].py are to be execute with python2 with pwntools installed.

code6.sh is a shell script and it requires "nmap" and "openssl" commands to be
installed.

code7.py uses a Mersenne twister predictor tool that needs to be installed by
the command "pip3 install mersenne-twister-predictor".

code8.py uses openssl's python module, install by "pip install pyopenssl".
